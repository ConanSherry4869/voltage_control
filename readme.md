
* @file voltage_control.cpp
* @brief 台区储能系统双向PI电压调节控制器
*
* 基于电压偏差的台区储能双向PI电压调节功能实现
* 功能描述：
* - 实时监测电网电压，根据电压偏差自动调节储能充放电功率
* - 过压时控制储能充电吸收无功功率，抑制电压升高
* - 欠压时控制储能放电提供无功支撑，提升电压水平
* - 具备SOC保护功能，防止电池过充过放
* - 支持JSON配置文件动态加载系统参数
